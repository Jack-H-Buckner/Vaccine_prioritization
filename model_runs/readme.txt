This folder has files that call and run the optimization rutines defined in the
code_model directory

There are four types of files in the directory

define_routines_**.jl - files that define function calls that produce desired data
Compare_outcomes_**.jl - files that compare outcomes when policies area applied to alternative distancing_senarios
run_optimization_**.jl - files that set parameters an call functions defined in the define_routines_**.jl files
error_bars_**.jl - files the run functions to generate error bars.
